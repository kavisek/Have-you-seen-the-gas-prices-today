# Vancouver Hackathon Research Notes

This folder is the Obsidian vault for the Vancouver Hackathon project.

## Structure

- Add your research notes as `.md` files in this folder
- Subdirectories are supported (e.g. `research/`, `ideas/`, `references/`)
- Notes sync with the GitHub repo via git

## Workflow

1. Write notes in Obsidian (saved locally as `.md` files here)
2. `git add notes/` → `git commit` → `git push` to sync to GitHub
3. Pull on other machines with `git pull`
